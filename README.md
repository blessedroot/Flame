## Использование

# maven

```xml
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
```

```xml
<dependencies>
<dependency>
	    <groupId>com.github.blessedroot</groupId>
	    <artifactId>Flame</artifactId>
	    <version>ВЕРСИЯ</version>
	</dependency>
</dependencies>
```

# gradle

```gradle
repositories {
	mavenCentral()
maven { url 'https://jitpack.io' }
		}
	}
```

```gradle
	dependencies {
	        implementation 'com.github.blessedroot:Flame:ВЕРСИЯ'
	}
```
